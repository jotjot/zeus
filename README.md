Zeus (0.0.1)
===============

Zeus is an RTMP Server that receives a streaming video from a Flash client and writes the video into an FLV file.


How to use it
---------------

Build it with:

```
./configure && make && make test
```

And run with:

```
./bin/zeus
```

The server will be run on ```rtmp://localhost:1935```


Required Libraries
-------------------

* libevent
